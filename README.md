## Getting Started

### Cloning the repository

```bash
git clone https://github.com/dhwanil005/ThinkBoardLab
```
### Installing Dependencies

```bash
npm i

```

### Setup .env file

``` bash
CONVEX_DEPLOYMENT=
NEXT_PUBLIC_CONVEX_URL=
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
LIVEBLOCKS_SECRET_KEY=
```
### Setup Convex

```bash
npx convex dev
```
### Start the app

```bash
npm run dev
```
